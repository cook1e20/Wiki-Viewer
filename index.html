A Wikipedia viewer

A Wikipedia viewer, another project from FreeCodeCamp, and another chance to test my knowledge of accessing APIs, in this challenge I created a small search engine, to search a key word and return articles relating to the word from Wikipedia. In this challenge I utilised the Wikipedia API, which can be found here, with a useful tool to create your own requests, however although it was useful it was very complex, but eventually I was able to find a solution and from this I could create a JSON request. Another part of the project was to include a random search function again using the Wikipedia API, and I also had to do some research into accessing the keystrokes, so I could start the search when the user pressed the enter key.
